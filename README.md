# 🛒 MVP E-commerce

Projeto **MVP de E-commerce** desenvolvido como **projeto pessoal de estudo**, com foco em **Backend**, arquitetura **REST** e boas práticas de organização de código. O objetivo é aplicar, na prática, conceitos aprendidos na graduação e em estudos independentes, simulando um sistema real de e-commerce.

---

## 🎯 Objetivo do Projeto

Desenvolver um **MVP de e-commerce backend** bem estruturado, escalável e reutilizável, que sirva tanto como **projeto de estudo** quanto como **base técnica para um produto comercial**.

Este projeto segue os princípios de clean code para facilitar a transição entre ambientes de prototipação e produção comercial,com foco em qualidade de código, segurança, escalabilidade e manutenibilidade.

Este repositório público representa a **fase educacional e de prototipação** do projeto.

---

## 🧠 Funcionalidades (em andamento)

* 📦 Cadastro e listagem de produtos
* 🔍 Busca de produto por ID
* 🧱 Estrutura modular de rotas e middlewares
* 🌐 API seguindo padrão REST
* 🧪 Validações básicas de dados

> Novas funcionalidades estão sendo implementadas continuamente.

## ▶️ Como Executar o Projeto

### Pré-requisitos

* Node.js instalado
* Gerenciador de pacotes (npm ou yarn)

### Passos

```bash
# Clone o repositório
git clone https://github.com/ViniciusAlmeidaAndrade/MVP-e-commerce.git

# Acesse a pasta do projeto
cd MVP-e-commerce

# Instale as dependências
npm install

# Inicie o servidor
npm start
```

O servidor será iniciado em ambiente local.
